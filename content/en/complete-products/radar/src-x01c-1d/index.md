## Product Positioning

SRC-X01C-1D is positioned for drone detection missions that require reliable target detection, practical deployment, and stable all-weather operation in civil security projects.

Typical use cases include Low-altitude security for borders, industrial sites, and energy facilities, Counter-UAS early warning for ports, airports, and coastal infrastructure, Rapid deployment radar support for temporary security operations, while the source material indicates a practical operating range of up to 1.5 km for the main surveillance task profile. The platform uses Two-Dim Phased Scanning to support continuous monitoring and target tracking.
