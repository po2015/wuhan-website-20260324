On May 16, 2025, we announced the next public set of Horizon platform components, extending the software roadmap beyond device management into operational workflow and system connectivity.

The newly announced module set includes **Horizon Fusion**, **Horizon Command**, **Horizon Archive**, and **Horizon Link**. We said these components are intended to support one connected operating environment in which target correlation, alarm handling, record continuity, and third-party integration can be managed as part of the same workflow.

We designed Horizon Fusion to correlate outputs from multiple sensing layers and reduce duplicated alarms. We position Horizon Command as the operator workspace for live monitoring and response coordination. Horizon Archive focuses on playback and event review, while Horizon Link is intended to support external interfaces with third-party systems and site infrastructure.

This announcement marks the next visible software milestone after the public introduction of Horizon Sense. More detail is available on our [Horizon page](/horizon/).
